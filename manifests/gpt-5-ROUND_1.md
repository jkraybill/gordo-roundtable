1) Core loop fun?
- Net: Yes, but it risks sameness by horse #3-5 unless the training mini-games and race fields stay fresh.
- Training: 5 mini-games × 20 sessions per horse is a lot of repetition. The 2-race cooldown helps, but I’ll burn out unless mini-games have variant modifiers (weather, track bias drills, “expert rules,” rotating layouts, risk/reward optional objectives).

- Racing: The whip/hold/encourage + proximity buffs is the fun anchor. Ensure pack behavior feels different per track/distance (pace pressure, traffic), or the loop flattens.
- Suggest: Inject variability every 3-4 races (exhibition challenges, pace scenarios, rival ghost, gate-draw challenges). If training can sometimes be swapped for “rest and vet” or “strategy seminar” with different small buffs, it breaks monotony.

2) Progression curve satisfying?
- 10 horses/200 races to first T1 win is borderline long for arcade, fine for home-cab if the generational jumps feel real.
- Lamarckian pass-through can make progress feel meaningful if every generation reliably steps up a class. If bad rolls + low Potential produce “dead” generations, it will feel grindy.
- Suggest:
  - Early-game compression: ensure horse #1 → #2 is a big leap (guaranteed better Potential band, or “Founder’s boon” after first championship).
  - Visible milestones: pedigree “Nick grade,” generational scorecard, and target projections so I feel the climb.
  - Allow early retirement without penalty if I smell a dud, banking partial training into genes (you do this, but reinforce it in UI and rewards).

3) Track network depth?
- This is interesting and unique. Planning residencies to seed future breeding routes is cool.
- Risk: Without excellent map UI (filters for tier, distance specialties, residency slots, unlocked/locked, championship prerequisites), it’ll feel like homework.
- Suggest: Route planner with pinboard: “Place a breeder at Ascot by S2” goal suggestions, highlight nearest qualifying opportunities, and show which future pairings become possible if I win X championship.

4) Name genetics metagame?
- I love the concept; it scratches the community-wiki itch. But one “god combo” in 4.78M makes it functionally irrelevant for most. If it’s too strong, it’ll warp the meta once discovered online.
- Track-locked pools keep discovery local, which is nice, but can also feel stingy.
- Suggest:
  - Keep bonuses meaningful but bounded so a cracked combo isn’t mandatory.
  - In-game rumor system: “Whispers” about name families (prefix/suffix hints) at each track, so discovery isn’t 100% external.
  - Slowly unlock cross-track name pools via championships to avoid running dry or feeling gated.

5) Championship qualification rules?
- They create good routing puzzles and purposeful wins. The friction is fine if failure isn’t punishing.
- Add safety valves: one wildcard per horse, or a points-based qualifier if you place consistently at the right tier but get blocked by draws/traffic.

- It’s focused and fits a cabinet. For 50+ hours, I’ll want some parallelism to break repetition and mitigate bad RNG.
